# Documentation

This directory contains stable project documentation. It intentionally avoids
development logs, one-off validation notes, and historical task checklists.

- [Usage](usage.md): installing the CLI, creating machines, and sending agent
  commands.
- [Architecture](architecture.md): host, runtime, guest agent, permissions, and
  protocol design.
- [Guest Image](guest-image.md): building the product and authorized macOS
  guest images.
- [Computer Use Plugin Parity](plugin-parity.md): implementation plan for
  aligning the CLI with the Computer Use plugin's app-scoped capabilities.
- [Releasing](releasing.md): signed macOS release artifacts and GitHub Actions
  secrets.
- [Development](development.md): local build, tests, smoke checks, and helper
  scripts.
